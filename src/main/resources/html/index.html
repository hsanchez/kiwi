The home of Kiwi-Vesper

Kiwi Vesper (code name vesper-http) was designed and developed by Huascar Sanchez. Its main purpose is to wrap Vesper, a code transformation library, and turn it into a RESTful service.

Reference sheet

Code Transformation Functionality

Supports a few code transformations, such as Safe delete, Clip code, Deduplication, Rename symbol (s), Unused Imports removal, Code formatting, etc.

Syntax verification

Detects whether the code snippet being edited has syntax errors.

Curated Code Presentation

Kiwi's second responsibility is to render individual HTML pages for any curated code snippets. These snippets are available, as tweets, on Twitter. So, if you have a Twitter account and want access to them, please follow @codedetour.
