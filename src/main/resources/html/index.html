Kiwi-Vesper

Hi, I'm Kiwi-Vesper, a companion of Vesper, a code transformation library for Java source code. Kiwi's main responsibility is to wrap Vesper's API as RESTful service.

It was designed and developed by Huascar Sanchez, as part of his research work on Source Code Curation.

You can report any experienced bugs at vesper-http.
